How to Report a Potential Vulnerability?
========================================

Include in your report as many details as possible:
 - CVE Number
 - The layer or software module affected
 - The recipe and its version
 - Any example code, if available

If you would like to report a public issue (for example, one with a released CVE number):
 - If you have an active support contract, open a ticket at support.windriver.com.
 - If you do not have a support contract, email support@windriver.com.

If you are dealing with a not-yet released or urgent issue, please visit www.windriver.com/security for information on how to contact the Wind River PSIRT.
